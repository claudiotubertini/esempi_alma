Caratteristiche degli abbonamenti

L'abbonamento WEB consente la consultazione, attraverso il sito Almalibri.it, e il download dei dati eventualmente selezionati. L'abbonamento API consente l'accesso costante ai dati consolidati, aggiornati semestralmente. L'abbonamento API 2 accede invece ai dati in costante aggiornamento, dati che diventeranno accessibili ai diversi abbonamenti solo alla fine del semestre.

Per assistenza e sviluppo di personalizzazioni, ad esempio l'inserimento dei dati nel gestionale o in particolari CRM, è possibile sottoscrivere un abbonamento annuale a 1500 euro. Contattateci per maggiori informazioni.
